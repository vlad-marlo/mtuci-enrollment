# mtuci-enrollment

## How to run:
Requirements
- Docker
- Docker compose
- compose.yaml file from this repo

### Steps:
1. get ```compose.yaml``` file on your computer
2. run ```docker compose up -d```
3. server will be running on localhost:8080
4. docs available on ```http://localhost:8080/docs```

## Technologies
- Docker
- Docker compose
- Python 3.11
- FastAPI
- Pydantic v2.0
- SQLAlchemy 2.0
- aiopg
- postgres 15
- structlog
